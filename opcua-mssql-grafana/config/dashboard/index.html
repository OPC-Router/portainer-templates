About This Sample

This sample demonstrates the integration of an Umati OPC UA server with an MSSQL database using the OPC Router. Data is visualized using Grafana, providing insights and monitoring capabilities.

The goal of this setup is to use the OPC Router to facilitate communication between the Umati OPC UA server and the MSSQL database. Grafana is used to create dynamic dashboards for data visualization.
